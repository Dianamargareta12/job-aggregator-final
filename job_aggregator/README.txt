JOB AGGREGATOR MODERN UI FINAL

Isi file:
1. index.php
2. style.css

Cara pakai:
1. Extract ZIP ini.
2. Copy index.php dan style.css ke folder:
   C:/xampp/htdocs/job_aggregator/frontend/
3. Timpa file lama jika diminta.
4. Jalankan Apache dan MySQL di XAMPP.
5. Buka:
   http://localhost/job_aggregator/frontend/index.php

Catatan penting:
- Database harus bernama: job_aggregator
- Tabel utama harus bernama: jobs
- Kode ini mendukung nama kolom judul_pekerjaan dan judul_posisi.
- Jika tampilan belum berubah, tekan CTRL + F5 pada browser.
